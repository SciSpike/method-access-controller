A regular expression identifying names of classes whose method invocations will be controlled.

A regular expression identifying names of methods on classes whose names match property classes.
strategy boolean | AccessControlStrategy | string

If strategy is one of the boolean literals true or false, then access is statically granted or denied, respectively. If strategy is an AccessControlStrategy, then it will be invoked to determine whether access is allowed. If strategy is a string, it is treated as a module, required, and expected to export a function of type AccessControlStrategy.
